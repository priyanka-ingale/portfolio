## üöÄ Quick Start

### Viewing Locally

1. Clone this repository:
```bash
git clone https://github.com/priyanka-ingale/priyanka-ingale.github.io.git
cd priyanka-ingale.github.io
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

### Deploying to GitHub Pages

1. Make sure your repository is named `priyanka-ingale.github.io` (or `<username>.github.io`)

2. Push the files to the main branch:
```bash
git add .
git commit -m "Initial portfolio commit"
git push origin main
```

3. Enable GitHub Pages:
   - Go to repository **Settings**
   - Navigate to **Pages** section
   - Under "Source", select **main** branch
   - Click **Save**

4. Your site will be live at: `https://priyanka-ingale.github.io`

## üé® Customization

### Updating Content

1. **Personal Information**: Edit the contact details in `index.html`:
   - Email, phone number, LinkedIn, GitHub links
   - Hero section text
   - About me section

2. **Experience**: Update timeline items in the Experience section
   - Add/remove positions
   - Update achievements and metrics

3. **Projects**: Modify project cards
   - Update project descriptions
   - Change technology tags

4. **Skills**: Add or remove skills in the Skills section

### Styling

- **Colors**: Modify CSS variables in `styles.css` under `:root`
  ```css
  :root {
      --color-primary: #1a1a2e;
      --color-accent: #e94560;
      /* ... other variables */
  }
  ```

- **Fonts**: Change font families in the Google Fonts link and CSS variables
  ```css
  --font-display: 'Your Display Font', serif;
  --font-body: 'Your Body Font', sans-serif;
  ```

- **Layout**: Adjust spacing, sizing, and responsive breakpoints
